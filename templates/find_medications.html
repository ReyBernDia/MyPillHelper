Identify a Pill

How to identify a pill using the Pill Identifier?

1. Enter an imprint code that appears on the pill e.g. APO

2. Select the score. The score is a groove to both indicate that a pill may be split and to aid the practice of splitting pills. If the pill does not appear to have a groove, select 1.

3. Select the shape (optional)

4. Select the the pill color (optional)
